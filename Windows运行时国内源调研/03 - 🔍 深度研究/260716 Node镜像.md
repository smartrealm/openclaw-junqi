# Node.js 国内镜像核验

**生成时间**：2026-07-16

## 核心结论

Node.js Windows ZIP 具备多家可直接使用的国内镜像。以当前 LTS `v24.18.0` 为样本，npmmirror、阿里云、腾讯云、中科大、南京大学和华为云均提供 `index.json`、x64/ARM64 发布文件与 `SHASUMS256.txt`。

## 可用镜像

- [npmmirror](https://npmmirror.com/mirrors/node/)：现有主源，目录完整并由国内 CDN 分发。
- [阿里云 nodejs-release](https://mirrors.aliyun.com/nodejs-release/)：发布文件和校验文件完整。
- [腾讯云 nodejs-release](https://mirrors.cloud.tencent.com/nodejs-release/)：发布文件和校验文件完整。
- [中科大 Node 镜像](https://mirrors.ustc.edu.cn/node/)：发布文件和校验文件完整。
- [南京大学 nodejs-release](https://mirror.nju.edu.cn/nodejs-release/)：发布文件和校验文件完整。
- [华为云 Node.js 镜像](https://mirrors.huaweicloud.com/nodejs/)：发布文件和校验文件完整。

六家镜像的 `v24.18.0/SHASUMS256.txt` 实测 SHA-256 完全一致。清华 TUNA 和北外虽然保留 `nodejs-release` 目录，但样本版本返回 404，不纳入运行时回退链。

## 应用建议

版本索引、校验文件和 ZIP 必须使用同一优先级列表；每个下载文件在激活前必须校验 SHA-256。终端用户安装不请求 `nodejs.org`。
