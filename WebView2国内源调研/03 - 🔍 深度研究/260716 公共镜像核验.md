# WebView2 公共镜像核验

**生成时间**：2026-07-16

## 核心结论

国内主流高校与云厂商公共开源镜像没有提供可直接用于 JunQi 安装器的 WebView2 Evergreen Runtime 仓库。常见 `/webview2/` 与 `/microsoft-edge/` 路径不能作为稳定源。

## 核验范围

- [阿里云开源镜像](https://mirrors.aliyun.com/)的公开目录没有 WebView2 或 Microsoft Edge Runtime。
- [腾讯云软件源清单](https://mirrors.cloud.tencent.com/source.js)公开的镜像项目没有 WebView2。
- [清华 TUNA](https://mirrors.tuna.tsinghua.edu.cn/legacy_index)、中科大、南京大学、北京外国语大学、上海交通大学、兰州大学及 [CERNET 联合镜像列表](https://mirrors.cernet.edu.cn/list)均未发现对应仓库。
- [华为云镜像 API](https://mirrors.huaweicloud.com/v1/repositories)列出的 135 个仓库包含 NuGet、MSYS2 和 Git for Windows，但没有 WebView2 Runtime；任意路径返回门户 SPA，不能视为文件镜像。

## 应用建议

不应把第三方软件下载站或猜测出来的镜像路径写入安装器。优先使用微软官方且在大陆落地 CDN 的分发链路。
