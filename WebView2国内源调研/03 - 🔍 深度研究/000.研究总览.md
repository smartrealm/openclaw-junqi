# WebView2 国内源深度研究总览

**生成时间**：2026-07-16

## 研究成果

| 课题 | 核心结论 |
| --- | --- |
| [公共镜像核验](./260716%20公共镜像核验.md) | 主流公共镜像没有 WebView2 Runtime |
| [微软大陆 CDN](./260716%20微软大陆CDN.md) | 官方分发链路已落地大陆 CDN，可直接使用 |
| [包管理渠道](./260716%20包管理渠道.md) | npm/NuGet 仅有 SDK 或无关包，不能替代 Runtime |

## 综合建议

采用微软官方 Bootstrapper + 大陆 CDN 的按需安装方案。主安装包保持轻量，不发布离线大包；系统已有 WebView2 时不产生额外下载，缺失时才获取 Runtime。
