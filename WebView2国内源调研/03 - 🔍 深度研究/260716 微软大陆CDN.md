# 微软 WebView2 大陆 CDN 核验

**生成时间**：2026-07-16

## 核心结论

Tauri 使用的微软官方 WebView2 Bootstrapper 可通过大陆 CDN 获取。官方入口 `https://go.microsoft.com/fwlink/p/?LinkId=2124703` 重定向到 `msedge.sf.dl.delivery.mp.microsoft.com`，引导程序约 1.61 MiB。

## 直链验证

- 最终域名的 DNS 链包含 `cdn.dnse8.com` 与 `tdnsdl1.cn`。
- 实测解析到 `1.82.220.65`、`1.82.220.66`、`1.194.175.186`、`121.29.13.56` 等大陆地址。
- HTTP Range 请求返回 `206`，并显示 CDN cache hit。
- [微软 winget WebView2 Runtime 清单](https://github.com/microsoft/winget-pkgs/tree/master/manifests/m/Microsoft/EdgeWebView2Runtime)同时提供 x86、x64、ARM64 官方安装器 URL 与 SHA-256。

## 应用建议

使用 Tauri `embedBootstrapper`：主安装包只增加约 1.6–1.8 MiB，系统已有 WebView2 时直接复用，缺失时才从微软大陆 CDN 下载运行时。不得改用约 127 MB 的 `offlineInstaller`。
