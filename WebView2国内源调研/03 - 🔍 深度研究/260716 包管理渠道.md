# WebView2 包管理渠道核验

**生成时间**：2026-07-16

## 核心结论

国内 npm 和 NuGet 镜像不能替代 WebView2 Runtime 分发。它们提供的主要是 SDK、Loader 或无关 JavaScript Runtime，不是微软 Evergreen Runtime 安装器。

## 核验结果

- npmmirror 与 npm 搜索没有可信的微软 WebView2 Runtime 安装器包。
- 华为云 NuGet 镜像可代理 `Microsoft.Web.WebView2` SDK，但 NuGet SDK 不等于系统 Runtime。
- winget 的 `Microsoft.EdgeWebView2Runtime` 清单是可靠的版本、架构、URL 和 SHA-256元数据来源，但安装器本身仍由微软 CDN 分发。

## 应用建议

安装器直接采用 Tauri 内置微软 Bootstrapper 流程；CI 可定期读取 winget 官方清单审计最新 Runtime URL 与 SHA-256，但不把 Runtime 复制进主安装包。
