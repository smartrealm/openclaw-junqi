# Gateway 单实例生命周期与存储迁移规格

## BUG-GL01/GL02 · 生命周期操作单飞

**当前**：restart、start、ensure、stop、Docker 各自管理并发。

**目标**：同一时刻最多一个生命周期写操作；后到的 ensure 等待并重新探测，不返回伪失败。

**验收**：

- [x] 并发获取生命周期门闩时只有一个所有者。
- [x] ensure 内部启动不会二次获取门闩而死锁。
- [x] restart 兜底启动与外部 start 不能并发。

## BUG-GL03 · 应用单实例

**当前**：多个 JunQi 进程可各自管理 Gateway。

**目标**：第二个应用实例不创建第二套生命周期管理器，只唤醒现有主窗口。

**验收**：

- [x] Tauri 注册单实例插件。
- [x] 第二实例事件显示并聚焦主窗口。

## BUG-GL04/GL05 · 统一状态所有权

**当前**：状态只可靠描述托管 Child。

**目标**：状态同时描述生命周期和运行模式，所有命令在成功、失败、停止时更新它。

**验收**：

- [x] start/restart/docker/stop 均产生明确模式转换。
- [x] 子进程退出后不会继续报告 managed_child/running。

## BUG-ST01 · bootstrap 路径解析

**当前**：固定使用 `~/.openclaw`。

**目标**：固定系统配置目录中的 bootstrap 决定状态、配置和工作区；显式 `OPENCLAW_*` 环境变量保持最高优先级。

**验收**：

- [x] 没有 bootstrap 时兼容 `~/.openclaw`。
- [x] bootstrap 生效后所有 `paths` 调用读取新目录。
- [x] bootstrap 本身不存放在可迁移状态目录中。

## BUG-ST02 · 首次启动迁移

**当前**：没有迁移入口。

**目标**：首次启动可继续使用默认目录、迁移到所选目录或创建新环境。

**验收**：

- [x] 迁移持有生命周期独占锁并停止受管进程。
- [x] 数据先复制到目标侧临时目录并完成文件数、总字节和主配置校验。
- [x] 只有校验成功才切换 bootstrap。
- [x] 失败时源目录和原 bootstrap 保持可用。
- [x] UI 在运行时检测后、安装执行前完成路径决策，并展示迁移进度与错误。
- [x] 存储步骤完成后恢复检测阶段选定的原有后续分支。

## BUG-ST03/ST04 · 停止确认与失败恢复

**当前**：停止命令按 best-effort 执行，复制或校验失败后原 Gateway 保持停止。

**目标**：只有端口确认释放后才复制；事务失败时恢复迁移前的运行方式并清理未提交目标。

**验收**：

- [x] 迁移前 Gateway 可达时，端口未释放会阻止复制。
- [x] managed child、Docker 和系统服务分别走对应恢复路径。
- [x] 文件事务或 bootstrap 失败时旧 bootstrap、源数据和原 Gateway 仍可用。
- [x] 失败目标被清理，用户可以对同一路径重试。

## BUG-ST05 · workspace 配置一致性

**当前**：迁移后的 workspace 路径补丁忽略错误，并在目标激活后执行。

**目标**：在临时目录中完成补丁，失败不激活目标。

**验收**：

- [x] 状态目录内部 workspace 被重写到新状态目录。
- [x] 外部 workspace 保持原路径。
- [x] 配置写入失败会清理临时目录并中止迁移。

## BUG-ST06 · Windows bootstrap 覆盖

**当前**：依赖 `rename` 覆盖已有 bootstrap。

**目标**：Windows 使用原生 replace-existing 语义，临时文件写入后同步落盘。

**验收**：

- [x] bootstrap 可以连续保存两个不同布局并读取最新值。
- [x] 写入失败不会遗留临时 bootstrap 文件。

## BUG-ST07 · 多语言迁移进度

**当前**：后台进度固定发送中文。

**目标**：事件携带稳定翻译 key，中文、英文、阿拉伯语由前端动态解析。

**验收**：

- [x] 进度事件包含 `key`、兜底 `message` 和归一化 `progress`。
- [x] 三种语言包含全部存储迁移阶段文案。

## BUG-ST08 · 符号链接状态根

**当前**：复制状态根链接会让临时目录仍指向源目录，配置补丁可能反写源文件。

**目标**：迁移状态根链接指向的实际目录内容，且源目录始终只读。

**验收**：

- [x] 状态根是符号链接时，目标是独立实体目录。
- [x] workspace 能按逻辑路径或规范路径正确映射。
- [x] 迁移补丁不修改源配置。
